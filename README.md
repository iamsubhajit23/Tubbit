## ğŸ“¦ Installation

### Prerequisites

- Node.js
- Express.js
- React.js
- MongoDB
- Cloudinary (for assests uploads)

### Clone the repo

```bash
git clone https://github.com/iamsubhajit23/Tubbit.git
cd Tubbit
```

### Setup Server

```bash
cd server
npm install
# Create a .env file with your MongoDB URI, JWT secret, and Cloudinary credentials
npm run dev
```

### Setup Client

```bash
cd client
npm install
npm run dev
```
